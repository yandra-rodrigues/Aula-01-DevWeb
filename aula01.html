O que é HTML?

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental usada para estruturar e criar páginas web. Ela define a organização do conteúdo, como títulos, parágrafos, imagens e links, agindo como o "esqueleto" de um site, interpretado pelos navegadores.
